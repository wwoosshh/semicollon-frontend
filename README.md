# semicolon-frontend

세미콜론 동아리 통합 플랫폼 — **프론트엔드**

## 스택
- Next.js (App Router) + TypeScript
- Tailwind CSS + shadcn/ui
- TanStack Query (서버 데이터)
- LiveKit React SDK (음성통화)
- Socket.IO client (실시간 채팅 · presence)
- PWA (윈도우/안드로이드/iOS/웹 단일 코드)

## 배포
- Vercel

## 연동
- API 서버: `semicolon-backend` (Railway)
- 음성: LiveKit Cloud
